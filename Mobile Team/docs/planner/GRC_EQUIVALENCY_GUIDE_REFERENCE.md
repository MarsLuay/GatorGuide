# GRC Equivalency Guide Reference

## Summary of this doc and what it contains

This is the single consolidated planner-facing reference doc for the Green River -> UW transfer and equivalency information the planner uses.
It now carries the migrated planner-relevant content that used to live in the separate raw markdown reference docs, so the six planner docs can stand on their own without depending on extra markdown files in `docs/reference`.

## What is combined here

- Green River transfer-track and latest published course-plan information.
- The cleaned current Green River -> UW equivalency guide used for planning.
- The detailed full-series and combo-credit rules that affect whether a course really counts the way a student expects.
- Planner-specific interpretation notes for how to use those rules safely.

## Planner-specific interpretation rules

- Prefer full-sequence outcomes over single-course guesses.
- Prefer current direct equivalencies over older workaround paths unless a target department still uses the older path in its own planning material.
- Treat Green River transfer tracks as launchpads, not perfect one-click major plans.
- Treat `No credit` rows as reference-only, not as planner recommendations.
- Keep a difference between `required before application`, `required before enrollment`, `worth finishing at Green River`, and `better saved for UW`.

## Important Reading Note

When this doc says "full credit rule" or "series rule," that means the exact combination UW says you must complete to receive the listed UW equivalency.

That is not always the same thing as a Green River catalog prerequisite.

Example:

- As of March 31, 2026, the UW Green River guide says `MATH& 153 (5)` alone transfers as `MATH 1XX`.
- The same current guide says `MATH& 153, 254 (5, 5)` transfers as `MATH 126, 224 (5, 4), 2XX (1)` if both courses are taken.
- So for current GRC coursework, the UW guide's transfer-combo rule for getting UW `MATH 126` is `MATH& 153 + MATH& 254`, not just `MATH& 153` by itself.
