# Green River -> UW Current Equivalency Guide

This is a cleaned working version of the Green River College -> University of Washington equivalency guide built from the user-provided UW page text.

- Included: current Green River course rows from the supplied guide text.
- Important: course prefixes with `&` matter. `ENGL& 101` and `ENGL 101` are not the same course.
- This is a planning/reference doc for Gator Guide. Official transfer decisions still belong to UW and Green River advisors.

## Cleanup Rules Used

- Preserved UW equivalency language like `1XX`, `2XX`, `[C]`, `[RSN]`, and `(LC)`.
- Grouped courses together when they share the same current UW equivalency.
