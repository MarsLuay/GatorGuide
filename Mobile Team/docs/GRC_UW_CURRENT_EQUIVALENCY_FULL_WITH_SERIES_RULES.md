# Green River -> UW Seattle Current Equivalencies (Full List + Series Rules)

Checked against the official UW Seattle Green River College equivalency guide on March 31, 2026.

Official source:

- https://admit.washington.edu/apply/transfer/equivalency-guide/green-river/

This doc is a current-only extraction of active Green River College courses that still appear in the UW Seattle equivalency guide.

- Included: current non-historical rows from the official UW guide.
- Omitted: rows marked with the section-sign on the UW site, because those are discontinued/past offerings.
- Omitted: current rows whose UW result is `No credit`.
- Preserved: UW labels like `UW 1XX`, `UW 2XX`, `A&H`, `SSc`, `NSc`, `[C]`, `[RSN]`, and `(LC)`.

## Important Reading Note

When this doc says "full credit rule" or "series rule," that means the exact combination UW says you must complete to receive the listed UW equivalency.

That is not always the same thing as a Green River catalog prerequisite.

Example:

- As of March 31, 2026, the UW Green River guide says `MATH& 153 (5)` alone transfers as `MATH 1XX`.
- The same current guide says `MATH& 153, 254 (5, 5)` transfers as `MATH 126, 224 (5, 4), 2XX (1)` if both courses are taken.
- So for current GRC coursework, the UW guide's transfer-combo rule for getting UW `MATH 126` is `MATH& 153 + MATH& 254`, not just `MATH& 153` by itself.

## Full-Credit / Series Rules You Should Not Miss

These are the current cases where the guide itself says you need a specific combination, total credit amount, or a full series to receive the stronger UW outcome.

### Accounting

- `ACCT& 201 + ACCT& 202`
  - Current UW rule: `ACCTG 215 (5), B A 2XX (5)` if both courses are taken.
  - Otherwise: `B A 2XX`.

### Astronomy

- `ASTR& 100` and `ASTR& 101`
  - Current UW rule: either course gives `ASTR 101 (5)`.
  - If both are taken, the second course gives `1XX (5)`.

### Biology

- `BIOL& 211 + BIOL& 212 + BIOL& 213`
  - Current UW rule: `BIOL 180, 200, 220, 2XX (5, 5, 5, 3)` if all three courses are taken.
  - Otherwise: `2XX`.

- `BIOL& 241 + BIOL& 242`
  - Current UW rule: `BIOL 118, 119 (5, 1), NURS 301 (4)` if both courses are taken.
  - Otherwise: `BIOL 2XX`.

### Chemistry

- `CHEM& 162 + CHEM& 163`
  - Current UW rule: `CHEM 152, 162 (5, 5), 1XX (2)` if both courses are taken.
  - Otherwise: `CHEM 1XX`.

- `CHEM& 261 + CHEM& 262 + CHEM& 263`
  - Current UW rule: `CHEM 237, 238, 239, 241, 242 (4, 4, 4, 3, 3)` if all three courses are taken.

- `CHEM& 261 + CHEM& 262`
  - Current UW rule: `CHEM 237, 238, 241 (4, 4, 3), 2XX (1)` if both courses are taken.

### Communication Studies

- `CMST 266`
  - Current UW rule: `CMS 272` if 5 credits are taken.
  - Otherwise: `CMS 2XX`.

### English

- `ENGL& 244 + ENGL& 245`
  - Current UW rule: `ENGL 250 (5)` for either course, plus `2XX (5-10)` if both courses are taken.

### History

- `HIST& 136 + HIST& 137`
  - Current UW rule: `HSTAA 101 (5), 1XX (5)` if both courses are taken.
  - Otherwise: `HSTAA 1XX`.

### Mathematics

- `MATH& 153`
  - Current UW rule: `MATH 1XX`.
  - By itself, it does not currently transfer as UW `MATH 126`.

- `MATH& 153 + MATH& 254`
  - Current UW rule as of March 31, 2026: `MATH 126, 224 (5, 4), 2XX (1)` if both courses are taken.
  - This is the current UW combination rule for receiving UW `MATH 126` from the Green River sequence listed in the guide.

### Natural Resources

- `NATRS 180 + NATRS 292`
  - Current UW rule: `ESRM 368 (4), ESRM 2XX (9)` for ESRM majors only.
  - Otherwise: see the individual course entries.
